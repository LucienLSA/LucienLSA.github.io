数组和字符串

入门题（字符串匹配）

Returns the position of the first occurrence of string target in string source, or -1 if target is not part of source.

假设在长度为n的母串中匹配长度为m的子串

Brute-Force算法：顺序遍历母串，将每个字符作为匹配的起始字符，判断是否匹配子串。时间复杂度O(m*n)

Rabin-Karp算法：将每个匹配子串映射为一个hash值。子串看作多进制数，比较它的值与母串中相同长度子串的hash值，如果相同，再细致地按字符确认字符串是否确实相同。顺序计算母串hash值的过程中，使用增量计算方法：扣除最高位的hash值，增加最低位的hash值。平均情况下算法复杂度为O(m+n)

Array

int array[arraySize]; //Stack上定义长度为arraySize的整型数组

int *array = new int[arraySize]; // Heap上定义长度为arraySize的整型数组，使用完后需要释放内存

delete[] array; // 定期回收释放内存

数组长度不定，则不能初始化数组。 int array[arraySize] = {0}; // 不定长度数组初始化为零

数组越界：访问不属于该数组的内存。数组名是指针，指向数组起始的地址。有些内存区只读，可能会造成崩溃，循环与传递数组需要注意

Stack and Heap

Stack

主要指由操作系统自动管理的内存空间。当进入一个函数，操作系统会为该函数中的局部变量分配存储空间。系统分配一个内存块，叠加在当前的stack上，并且利用指针指向前一个内存块地址。

函数的局部变量存储在当前的内存块上，当函数返回时，系统弹出内存块，并且根据指针回到前一个内存块。Stack总是以后进先出的方式工作

Heap

Heapu用来存储动态分配变量的空间，随时分配或回收内存。程序员自己命令回收内存，否则会产生内存泄漏

heap 创建

Vector容器

Hash Table

几乎最重要的数据结构，基于key的查找，存储的基本元素为key-value的pair。key是一个非负整数

碰撞

  1. Open Hashing
    持续散列方式，直到产生的下标对应还未使用存储的地址。

  2. Closed Hashing
    共享同一个hash table 位置时，将后一个放在前一个的后面，需要附件的空间。

C++标准库

提供map容器，可插入删除和查找key-value pair，底层以平衡二叉搜索树的方式实现，根据key进行排序。

c++11中，标准库添加了unordered_map，更符合Hash Table的传统定义，平均查找时间O(1)

String类

字符串指一个以’\0’结尾的char数组，关于字符串的函数需要传入一个字符型指针。

String是一个类，通过调用类函数实现判断字符串的长度、字符串等等

模式识别

统计一个元素集中元素出现的次数，使用Hash Table。可使用std::unordered_map 或 std::map: key是元素，value是出现的次数。有些仅需要判断元素出现与否（相当于判断value是0或1），可以用bitvector，利用一个bit来表示当前的下标是否有值

处理当前节点需要依赖之前部分结果，可考虑hash table 记录之前的处理结果

Example1

Determine if all characters of a string are unique

unique出现，使用hash table或bitset来判断元素出现

Example2

Give two strings, determine if they are permutations of each other

置换：无论如何变化，每个字符出现的次数一定相同，一旦需要统计一个元素集中元素出现的次数，应该用hash table

hash table 需扫描整个string，平均时间复杂度都是O(n)，比较两个hash是否相同，每个合法字符都可能出现，假设字符集大小为m，则需要时间复杂度为O(m)，总的时间复杂度O(m+n)，空间上平均空间复杂度为O(m)

解法2：对每个string中的字符按照ASCII编码顺序进行排序，置换则排序完的两个string应该相等，时间复杂度为O(nlogn)，空间复杂度为O(n)

Example3

Give a newspaper and message as two strings, check if the message can be composed using letters in the newspaper

message中用到的字符必须出现在newspaper中，message中任意字符出现的次数一定小于其在newspaper中出现的次数。统计一个元素集中元素出现的次数，用hash table

假设message长度为m，newspaper长度为n，算法需要hash整个message和newspaper，时间复杂度O(m+n)，假设字符集大小为c，平均空间是O©

Anagram

Write a method anagram(s,t) to decide if two strings are anagrams or not

Given s = “abcd”, t = “dcab”, return true

Example4

Find a pair of two elements in an array, whose sum is a given target number

再次扫描数组，判断target array[i] 是否存在数组中，这样时间复杂度是O(n^2)

Example5

Get the length of the longest consecutive elements sequence in an array. For example, give [31, 6, 32, 1, 3, 2], the longest consecutive elements sequence is [1,2,3]. Return its length3

判断array[i]-1, array[i]+1，是否存在数组中。如何保存之前处理的结果，用hash table。序列的最小值和最大值，能唯一确定序列。作为后继加入序列，作为前驱加入序列，更新最大最小值。hash table的value可以是记录最大值/最小值的structure，用以描述当前节点参与构成的最长序列

Longest Common Substring

Give two strings, find the longest common substring. Return the length of it. Given A=“ABCD”, B=“CBCE”, return 2. The characters in substring should occur continuously in original string.
